

<h2>1.	Implemente uma codificação para:</h2>
<ul>
<li>     a.	Definição e criação de uma estrutura de dados que representa um aluno contendo:</li>
<br> - i.	Matricula do tipo inteiro;
<br> - ii.	Nome do tipo string;
<br> - iii.	Nota1, Nota2 e Nota3 do tipo double;
<br>
<br>
<li> b.	Dados para 3 alunos deverão ser informados (Informe notas entre 0 e 10).</li>
<li> c.	Criar uma função que retorne a matrícula do aluno com a maior Nota1.</li>
<li> d.	Criar uma função que retorne o nome do aluno com maior média das notas.</li>
<li> e.	Criar uma função que retorne o nome do aluno com menor média das notas.</li>
<li> f.	Criar um procedimento que informe se o aluno foi aprovado ou reprovado, considerando que a média é 6;</li>

<h2>2.	Implemente uma codificação para:</h2>
<ul>
<li>a.	Definição e criação de três estruturas de dados:</li>
<br> - i.	Hora: que terá três inteiros = hora, minutos e segundos.
<br> - ii.	Data: que terá três inteiros = dia, mes e ano.
<br> - iii.	Agendamento: que terá um tipo hora, um tipo data e um de caracteres para informar a descrição do que foi agendado.<br>
<li>b.	Crie um vetor agendamento de 3 posições.</li>
<li>c.	Crie uma função ou procedimento para informar 3 agendamentos.</li>
<li>d.	Crie uma função ou procedimento para mostrar os agendamentos na tela.</li>
</ul>

<h2>3.	Implemente uma codificação para:</h2>
<ul>
<li>a.	Definição e criação de uma estrutura que represente uma banda contendo:</li>
<br> - i.	Nome e gênero musical do tipo caractere.
<br> - ii.	Número de integrantes e ranking do tipo inteiro.
<br> - iii.	Ranking seria a preferência dentre as suas 5 bandas favoritas (informar entre 1 e 5).<br><br>
<li>b.	Crie um procedimento para informar 5 bandas.</li>
<li>c.	Crie um procedimento para exibir todas as informações das bandas.</li> 
<li>d.	Crie uma função que receberá o ranking da banda e exibirá as informações da banda cuja posição no seu ranking é a que foi solicitada pelo usuário.</li>
<li>e.	Crie um procedimento que receba o nome de uma banda e retornará se esta faz parte de suas bandas favoritas ou não.</li>
</ul>
